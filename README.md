# Multifractal Analysis of Earthquake Inter-Event Time Series

## Overview

This repository investigates the multifractal characteristics of **inter-event time series** derived from the **Japan Global Earthquake Catalog**. The analysis aims to quantify the long-range temporal correlations and complexity of earthquake occurrence using classical and multifractal techniques.

## Methodology

The workflow includes:

- Extraction of earthquake inter-event time series
- Estimation of the **Hurst Exponent** using the **Rescaled Range (R/S) method**
- Multifractal Detrended Fluctuation Analysis (**MFDFA**)
- Interpretation of multifractal parameters and singularity spectrum


## Results

The analysis reveals that the earthquake inter-event time series exhibits **persistent long-range correlations** and **strong multifractal characteristics**, indicating that the temporal evolution of seismicity is governed by complex, heterogeneous processes rather than purely random behavior.
